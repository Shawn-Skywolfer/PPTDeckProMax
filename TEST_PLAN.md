# PPT Deck Pro Max — 完整测试计划

## 测试环境要求

- 浏览器：Chrome / Edge / Firefox 最新版
- 本地服务器：`python start_server.py` 运行在 `http://localhost:8080`
- 网络：可访问外部 LLM API（AIHubMix / DeepSeek / OpenAI 等）
- 测试数据：准备一段中文/英文源材料（500-2000 字）
